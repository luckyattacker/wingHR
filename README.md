# wingHR 🚀

AI-powered HR Onboarding Wingman designed to automate the heavy lifting of onboarding new employees.

## Features
- ✅ **Typeform Integration**: Automatically pulls new hire data.
- ✅ **Slack Automation**: Sends workspace invites to new joiners.
- ✅ **Gmail Automation**: Automatically requests access for VPN, Kantata, and Internal Databases.
- ✅ **Automated Form Filling**: Browser-based automation for internal company portals.

## How it works
1. A new joiner fills out the onboarding Typeform.
2. wingHR captures the details and triggers a series of API calls.
3. The new hire receives their Slack invite and IT gets their access request immediately.
